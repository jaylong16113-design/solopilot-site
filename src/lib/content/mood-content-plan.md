# 情绪短视频板块 - 内容规划

## 方向总结

根据 Ricky 的创作经验和爆款复盘，情绪短视频的5个核心方向：

---

## 方向1：微表情 + 情绪转折（3微表情=1个情绪转折）

**核心逻辑**：情绪短视频的爆发力不靠长铺垫，而靠3个微表情串联出一个情绪转折。

**文章选题：**
- [ ] 微表情入门：3个表情如何织出一个"意难平"的故事
- [ ] 从面无表情到崩溃——用4个微表情镜头撑起15秒爆发
- [ ] Silent Diary 的微表情密码：眼神→嘴角→肩膀，他们怎么让你上头
- [ ] AI能模拟微表情吗？HeYGen/D-ID/阿里数字人生成微表情实测
- [ ] 情绪转折的9种公式：从"喜悦→隐瞒"到"坚强→崩塌"

---

## 方向2：AI短视频去AI化（让AI生成的视频更像真人）

**核心逻辑**：AI生成的视频一眼假是最大痛点，需要技术+技巧去AI味。

**文章选题：**
- [ ] AI数字人怎么做才不像机器人？5个让你一眼真人的技巧
- [ ] 面部表情去AI化：HeYGen/D-ID/Fish Audio哪个微表情最自然
- [ ] 声音去AI化：Silent Diary 的配音是怎么做到不像TTS的
- [ ] 眼神、眨眼、嘴唇不同步——AI短视频最常见的7个穿帮
- [ ] 实测：把AI生成的视频用剪映再"人工"一遍，去AI味的5个关键步骤
- [ ] 光线、噪点、背景——AI视频"太干净"怎么破？

---

## 方向3：演员级的Prompt工程（用Prompt写出"表演感"）

**核心逻辑**：AI生成的脚本往往空洞——要写出"有表演空间"的脚本。

**文章选题：**
- [ ] 脚本Prompt的进阶写法：不要"描述动作"，要"描述情绪"
- [ ] Silent Diary 的脚本结构拆解——用Prompt还原他们的"留白"艺术
- [ ] 表演级Prompt模板：给ChatGPT/Claude的12个情绪引导词
- [ ] 从"她哭了"到"眼眶先红→强忍→低头→肩膀抖→终于哭出声"——分镜级Prompt写法
- [ ] 情绪浓度量化：用"沉默秒数"和"呼吸节奏"写进Prompt

---

## 方向4：镜头语言（用构图和运镜讲故事）

**核心逻辑**：情绪短视频的本质是"用镜头说话"，普通人也能学会。

**文章选题：**
- [ ] 情绪短视频的镜头语言入门：特写、浅焦、晃动的情绪密码
- [ ] 想要Silent Diary的质感？5个手机运镜技巧+剪映调色参数
- [ ] 暖色调 vs 冷色调——情绪短视频的色彩心理学
- [ ] 空镜头不是废镜头：15秒视频里3秒空镜头，情绪直接拉满
- [ ] 灯光入门：在家用一盏台灯拍出电影级情绪光影
- [ ] 慢动作 + 环境音：Silent Diary最常用的两个技巧

---

## 方向5：对标账号分析（Silent Diary 深度拆解）

**核心串联方向**：每篇文章都引导用户搜索【Silent Diary】作为实战案例。

**文章选题：**
- [ ] 【Silent Diary】是什么？为什么每条都能刷哭你——一个账号的创作密码
- [ ] 拆解 Silent Diary 的3种经典叙事结构
- [ ] 从 Silent Diary 学配乐：BGM是情绪的放大器，不是背景音
- [ ] Silent Diary 的30秒结构：前10秒代入→中间15秒展现→最后5秒反转/余韵
- [ ] 除了 Silent Diary，小红书/抖音/TikTok上还有哪些不可错过的情绪短视频账号
- [ ] Silent Diary 教你：不需要露脸也能靠情绪出圈
