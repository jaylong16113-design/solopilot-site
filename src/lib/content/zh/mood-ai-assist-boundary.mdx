# AI辅助创作的边界：我自己用AI工具做情绪短视频的真实经验

我每天至少用4个AI工具处理短视频相关工作。但在情绪内容这块，我踩过很多坑。

最惨的一次：我用AI生成了完整的分镜脚本，配了AI生成的旁白，用AI绘画做了背景图——全程无人干预。发出去后，评论区最火的评论是这看着好假。

## AI擅长：不需要真情感的环节

1. 选题和关键词发现：每天跑一遍小红书和抖音的热搜，让AI分析出今天大家最关注的情绪关键词。工具推荐：opencli + ChatGPT

2. 文案草稿和标题优化：让AI分析Silent Diary过去100条标题的句式结构，生成20个相似但不相同的标题。工具推荐：Claude

3. 剪辑辅助：AI分析完播率高的视频，标记出情绪积累段和情绪释放段的分界点，生成通用剪辑节奏模板。

## AI不擅长：需要真实感的环节

1. 镜头语言设计：AI可以告诉你用特写表现情绪，但它不知道什么样的光线、角度能产生真实的情感冲击。
2. 配音和旁白：情绪内容的核心是微妙的停顿和气息，目前只有真人能做。
3. 画面筛选：AI选的全是表情最清晰的，但不是最自然的。这部分很难AI化。

## 我现在的工作流

选题 -> AI生成5个关键词 -> 我从中选1个
文案 -> AI生成10个标题 -> 我选3个组合修改
分镜 -> 我自己手写（不用AI）
拍摄 -> 纯人工
剪辑 -> AI做节奏模板 -> 我自己微调
配音 -> 纯人工
发布 -> AI生成标题优化方案 -> 我选最优
数据 -> AI分析优化建议

核心原则：AI做数据分析和重复工作，人做创意决策和情感把控。
